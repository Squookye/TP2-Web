Exercice 2 : Structuration du texte SQL

1. Le langage SQL

1.1. Définition

Le langage SQL (Structured Query Language) est un langage comprenant l'ensemble des ordres nécessaires à la création et à la gestion d'une base de données relationnelle.

1.2. Les différentes normes utilisées

Il existe aujourd'hui plusieurs normes relatives à l'utilisation du langage SQL :

  • La norme SQL 89
  • La norme SQL 92
  • Aujourd'hui SQL 2011

Note : Bien qu'il existe des normes, il peut y avoir des différences entre SGBDR (PostgreSQL, MySQL, etc.).
