Langage JavaScript : Les Bases

Sommaire

  • 1. Structure de base
  • 2. Les Variables (let, const, var)
  • 3. Les Types de données

1. Structure de base

Le JavaScript s'intègre dans le HTML via la balise <script>.

<script>
  // Les instructions JavaScript ici
  let message = "Bonjour !";
  alert(message);
</script>

2. Les Variables

Il existe trois façons de déclarer des variables :

  • let : Portée locale (recommandé).
  • var : Portée globale.
  • const : Pour les valeurs qui ne changent pas.

3. Les Types de données

JavaScript utilise plusieurs types primitifs :

  • Number : Entiers et réels.
  • String : Chaînes de caractères.
  • Boolean : true ou false.
  • BigInt : Pour les très grands entiers.
